Add or Remove a Scene

Step 1: Select the scene you want to embed in your post or pag.

Step 2: If your blog pages or posts reside withing a secure web environment check the Secure Protocol (HTTPS) checkbox.If you want the scene to playback each time the Web page is loaded, check the Play on Load checkbox. If you want to play or pause the scene by clicking the character in the scene, check Play by Click checkbox.

Step 3: Scene Dimensions and Background Color

Scene Dimensions: The suggested dimensions are 400 x 300 pixels. With the Keep Scene Dimensional Proportional check box selected, as you type a width, the height is automatically calculated so that it appears proportionally to the specified width.
Background Color:

Click the color selector button and select a color or Transparent from the palette. You can also enter the hexadecimal value for a color in the box centered below the palette.

Note: The background color that you set is the background color if you selected no background for a Scene on the Background Selection page.


Step 4: Click the Add To Post button to add your scene to your post or page.
